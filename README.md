# 🌐 Jomari R. Realon — Personal Portfolio Website

A modern, glassmorphic dark-themed personal portfolio website built with **React + Vite**. This site showcases my background, featured projects, technical skills, and contact information in a premium, responsive single-page application.

---

## 👤 About

**Jomari R. Realon** — Computer Science Graduate with an Intelligent Systems major. I specialize in building intelligent, high-performance applications with hands-on experience in Mobile App Development (Flutter) and Machine Learning model integration.

## ✨ Features

- **Glassmorphism dark theme** with a curated Space Indigo color palette
- **Smooth animations** — fade-in, glow pulses, hover micro-interactions
- **Fully responsive** — optimized for desktop, tablet, and mobile
- **Orientation-aware image gallery** — portrait images display side-by-side, landscape images stack vertically
- **Sections:** Hero · About & Education · Experience · Projects · Skills & Interests · Contact

## 📦 Getting Started

### Prerequisites
- Node.js ≥ 18
- npm ≥ 9

### Install & Run

```bash
# Clone the repository
git clone https://github.com/blurp0/portfolio-website.git
cd portfolio-website

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be available at **http://localhost:5173**

### Build for Production

```bash
npm run build
```

The output will be in the `dist/` folder, ready for deployment.
